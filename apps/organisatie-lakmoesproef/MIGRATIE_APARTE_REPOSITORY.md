# Deze app hoort niet langer in de Moral Maps-monorepo

De **organisatie lakmoesproef** wordt gescheiden van `moral-maps` / Moral APS.

## Wat je doet

1. Kopieer **deze hele map** (`organisatie-lakmoesproef`) naar een nieuwe projectmap op je computer.
2. Initialiseer daar een **eigen** git-repository en push naar je **eigen** GitHub-repo (bijv. `organisatie-lakmoesproef` of `morele-lakmoesproef`).
3. In Vercel: **Root Directory = `.`** (niet `apps/...`).
4. Verwijder deze map uit de monorepo met:
   ```bash
   git rm -r apps/organisatie-lakmoesproef
   ```

Volledige stappenlijst: **`../../docs/SCHEIDING_MAPS_EN_ORGANISATIE_LAKMOES.md`**
