# Moral Maps 3 - Je ware koers

Deel 3 van de MAPS-trilogie: Je ware koers.

## Stack

- Next.js App Router
- TypeScript
- Tailwind CSS
- Framer Motion

## Starten

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Huidige scope

- Signature-opdracht: Brug in de Mist
- Terugblik en vooruitblik
- Syntheseveld
- Lokale autosave via `localStorage`
