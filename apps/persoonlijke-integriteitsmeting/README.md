# Persoonlijke integriteitsmeting

Professionele individuele reflectie-app voor moreel handelen.

## Stack
- React + Vite
- TypeScript
- Tailwind CSS

## Scripts
- `npm run dev`
- `npm run build`
- `npm run lint`
- `npm run preview`

## Vercel
- Root directory: `apps/persoonlijke-integriteitsmeting`
- Build command: `npm run build`
- Output directory: `dist`
