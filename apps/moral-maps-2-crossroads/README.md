# Moral Maps 2 - Crossroads

Educatieve reflectie-app waarin deelnemers per opdracht hun morele koers verkennen.

## Starten

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Build en lint

```bash
npm run build
npm run lint
```

## Vercel

- **Root Directory**: `apps/moral-maps-2-crossroads`
- **Build Command**: `npm run build`
- **Output Directory**: `.next`

## Standaard en templates

- Standaard app structuur: `APP_STANDARD.md`
- Quick start (kort): `APP_QUICKSTART.md`
- Migratie-template: `APP_MIGRATION_TEMPLATE.md`

Gebruik deze drie documenten als vaste basis voor:
- nieuwe Moral Maps apps
- verbouwen van bestaande apps naar dezelfde structuur en stijl

## Kernprincipes

- Next.js App Router + TypeScript
- 1 opdracht per pagina
- eigen afbeelding per opdracht
- autosave en voortgang
- rustige, consistente UI (Apple-like)
